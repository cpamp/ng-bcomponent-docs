Ng-bcomponents is an Angular 2 component library of Bootstrap 3 components intending to make HTML markup simpler. Ng-bcomponents also allows you initialize bcomponents in code.

Ng-bcomponents is great for making HTML bootstrap markup simpler. Take the modal for example:

Modal Demo Modals sure are simple with ng-bcomponents!

The markup for this demo looks like this:

<button-bcomponent text="Show modal" type="primary" [click]="demoModal.open"></button-bcomponent>
<modal-bcomponent #demoModal>
    <modal-title>Modal Demo</modal-title>
    <modal-body>Modals sure are simple with ng-bcomponents!</modal-body>
    <modal-footer>
        <button-bcomponent text="OK" type="primary" [click]="demoModal.close"></button-bcomponent>
        <button-bcomponent text="Cancel" type="default" [click]="demoModal.close"></button-bcomponent>
    </modal-footer>
</modal-bcomponent>
