All BComponents extend the BComponent class and as a result have the following API as well.

disableAutoIdentifier() is used to disable automatic generation of id attributes. If you specify an id for a BComponent then an auto identifier will not be used for that BComponent. Call this method in the constructor of your NgModule to disable auto identifiers. 

public static disableAutoIdentifier: () => void

InitializeAttributes() is used to intialize components in code. Used to assign attribute inputs to a BComponent. Returns itself for method chaining. 

public InitializeAttributes: (id: string = "", classes: string = "", styles: string = "", name: string = "", aria: string = "", ariaBy: string = "") => this

getSelector() returns a JQuery object of the BComponent.
Note: This and all the following JQuery animation methods only work with auto identifiers enabled or a specified id. 

public getSelector: () => JQuery

toggle() is the equivalent of jQuery's toggle().

  

public toggle: (duration: string | number = 400, callback?: Function) => JQuery

hide() is the equivalent of jQuery's hide().

  

public hide: (duration: string | number = 400, callback?: Function) => JQuery

show() is the equivalent of jQuery's show().

  

public show: (duration: string | number = 400, callback?: Function) => JQuery

fadeToggle() is the equivalent of jQuery's fadeToggle().

  

public fadeToggle: (duration: string | number = 400, callback?: Function) => JQuery

fadeIn() is the equivalent of jQuery's fadeIn().

  

public fadeIn: (duration: string | number = 400, callback?: Function) => JQuery

fadeOut() is the equivalent of jQuery's fadeOut().

  

public fadeOut: (duration: string | number = 400, callback?: Function) => JQuery

fadeTo() is the equivalent of jQuery's fadeTo().

  

public fadeTo: (duration: string | number = 400, opacity: number = 0, callback?: Function) => JQuery

slideToggle() is the equivalent of jQuery's slideToggle().

  

public slideToggle: (duration: string | number = 400, callback?: Function) => JQuery

slideDown() is the equivalent of jQuery's slideDown().

  

public slideDown: (duration: string | number = 400, callback?: Function) => JQuery

slideUp() is the equivalent of jQuery's slideUp().

  

public slideUp: (duration: string | number = 400, callback?: Function) => JQuery
