All BComponents extend the BComponent class and as a result have all of the following inputs as well. HTML attributes are, in most cases, applied to the top level element in a component. 

{{code.overview}}

The id input is used to assign an id attribute to a bcomponent.
Example: 

{{code.id}}

The classes input is used to assign classes to a class attribute of a bcomponent. 

{{code.classes}}

The styles input is used to assign styles to a style attribute of a bcomponent. 

{{code.styles}}

The name input is used to assign a value to a name attribute of a bcomponent. 

{{code.name}}

The aria input is used to assign a value to the aria attribute of a bcomponent. 

{{code.aria}}

The ariaBy input is used to assign a value to the aria-labelledby attribute of a bcomponent. 

{{code.ariaBy}}

The bcomponent input is used to initialize bcomponents in code. All of the properties of the passed BComponent copied are to the BComponent to which it was passed. 

{{code.bcomponent}}
